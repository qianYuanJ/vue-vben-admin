# 仪表盘模块

## 模块概述

仪表盘模块是系统的首页，为用户提供系统核心数据的可视化展示和快速访问常用功能的入口。该模块包含数据分析和工作台两个子页面，分别面向不同的使用场景。

## 页面结构

### 数据分析页 - `apps/web-antd/src/views/dashboard/analytics/index.vue`

数据分析页主要展示系统的统计数据和趋势图表，帮助管理员了解系统整体运行状况。

**主要组件：**

- 核心指标卡片区域：展示用户数、订单数、访问量等关键指标
- 趋势图表区域：通过折线图、柱状图等可视化方式展示数据趋势
- 排行榜区域：展示热门内容、活跃用户等排名数据

### 工作台页 - `apps/web-antd/src/views/dashboard/workspace/index.vue`

工作台页面主要为用户提供日常工作所需的信息和快捷入口，提高工作效率。

**主要组件：**

- 用户信息区：展示当前用户的基本信息和工作状态
- 待办事项区：显示用户待处理的任务和提醒
- 快速入口区：提供常用功能的快速访问入口
- 通知公告区：展示系统公告和消息提醒
- 团队动态区：展示团队成员的近期活动

## 数据交互

### API接口

- 获取核心指标数据：`GET /api/dashboard/stats`
- 获取趋势图表数据：`GET /api/dashboard/trends`
- 获取排行榜数据：`GET /api/dashboard/rankings`
- 获取待办事项：`GET /api/dashboard/todos`
- 获取通知公告：`GET /api/dashboard/notices`
- 获取团队动态：`GET /api/dashboard/team-activities`

### 数据刷新策略

- 页面加载时自动获取最新数据
- 支持手动刷新功能
- 可配置自动刷新间隔（默认关闭）

## 关键实现

### 数据可视化

使用 ECharts 图表库实现数据可视化：

1. 封装通用图表组件，支持响应式调整
2. 提供多种图表类型：折线图、柱状图、饼图等
3. 支持主题切换，适应系统暗色/亮色模式

### 卡片组件

采用卡片式布局组织内容：

1. 统一的卡片组件，包含标题、内容、操作区
2. 支持拖拽排序（可选功能）
3. 支持展开/折叠状态记忆

### 响应式设计

为不同尺寸的设备提供良好的展示效果：

1. 使用栅格布局，根据屏幕宽度自动调整布局
2. 小屏幕设备优先展示核心信息
3. 图表组件自适应容器尺寸

## 自定义配置

仪表盘支持一定程度的个性化定制：

1. 用户可以选择显示/隐藏特定区块
2. 支持调整部分组件的展示顺序
3. 可设置数据自动刷新的频率

## 注意事项

1. 仪表盘页面数据量较大，应注意优化加载性能
2. 图表组件应考虑浏览器兼容性和低配置设备的性能
3. 敏感数据应根据用户权限进行展示控制
4. 实时数据更新应考虑服务器负载，合理设置请求频率
